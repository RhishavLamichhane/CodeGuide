Hypertext Markup Language (HTML)

HTML stands for Hyper Text Markup Language. It is the standard markup language for creating Web pages. HTML describes the structure of a Web page and consists of a series of elements that tell the browser how to display the content. HTML is easy to learn and is used to create static web pages and dynamic web applications.
